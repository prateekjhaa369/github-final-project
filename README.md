# github-final-project

## Simple Interest Calculator
A Bash script that calculates simple interest based on user input for principal, annual rate of interest, and time period in years.

### How to Use
1. **Clone the repository**: `git clone https://github.com/prateekjhaa369/github-final-project.git`
2. **Navigate to the directory**: `cd github-final-project`
3. **Make the script executable**: `chmod +x simple-interest.sh`
4. **Run the script**: `./simple-interest.sh`

### Input Fields:
- `p`: Principal amount (e.g., 1000)
- `r`: Annual rate of interest (e.g., 5 for 5%)
- `t`: Time period in years (e.g., 2)

### Output:
- The script will display the calculated Simple Interest.

### Example:
If you enter 1000 for principal, 5 for rate, and 2 for time, the output will be 100.
